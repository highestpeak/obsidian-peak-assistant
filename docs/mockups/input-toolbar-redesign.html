Key Insights

Pattern 1: 2-3 items max in toolbar. ChatGPT/Claude/Gemini all have 2-3 visible controls. Complexity lives in popovers.

Pattern 2: Mode is prominent. ChatGPT "Tools" dropdown, Cursor tabs — mode fundamentally changes behavior so it deserves visual weight.

Pattern 3: Model is text, not icon. Every product shows the model name as readable text, not hidden behind an icon.

Pattern 4: Settings are rare-use. Output control (temperature, etc.) is a power-user feature. ChatGPT/Claude hide it entirely from the input area. TypingMind puts it behind a single gear icon.

Your current design has 8 items. The industry standard is 2-4. The redesign should consolidate.
