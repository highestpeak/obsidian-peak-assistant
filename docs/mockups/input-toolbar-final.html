Design Summary

Layout: A dock 的 3 icon 左置 + 右侧 context info，mode 从 segmented 改为 pill+popover。

4 功能分区（每个一个 popover）：
  [+] Attach — upload mode 选择
  [🔍] Sources — vault/web toggle + code interpreter toggle（合并了原来的 search + tools 两个按钮）
  [⚙] Settings — output control 滑块（creativity / reasoning / detail）
  [Mode ▾] — chat/plan/agent popover（icon + 标题 + 描述，和你 Image 3 的设计一致）

右侧信息： mode pill + model badge (monospace 文字) + token count (仅宽屏)

窄侧栏适配： 隐藏 token count，截断 model name，所有功能保留
